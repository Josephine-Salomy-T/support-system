# Support System

A full-featured **Support Ticketing System** built with **React, Node.js, Express, and MongoDB**.  
Designed to help manage customer tickets, assign agents, and track ticket status in real-time.

---

##  Features

- Create, update, and delete tickets
- Assign tickets to agents
- Filter and sort tickets by status, priority, or agent
- Real-time updates for ticket assignments using Socket.io
- Authentication with JWT
- Responsive and user-friendly dashboard

---

##  Tech Stack

### Frontend
- React, TypeScript
- Vite
- Axios
- Chart.js & react-chartjs-2

### Backend
- Node.js, Express
- MongoDB, Mongoose
- JWT authentication
- Socket.io for real-time updates
- bcryptjs for password hashing
